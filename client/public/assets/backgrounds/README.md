# Custom scene backgrounds

Drop an image in here using these exact filenames to replace the default
procedural corkboard texture for that screen. If a file isn't present, the
app quietly falls back to the CSS cork texture — nothing breaks.

| Filename        | Used on                                  | Suggested size        |
|-----------------|-------------------------------------------|------------------------|
| `lobby.jpg`     | Landing page + "waiting in lobby" screens | 1920x1080, landscape   |
| `writer.jpg`    | Job Seeker's resume-writing screen        | 1920x1080, landscape   |
| `manager.jpg`   | Manager voting screen                     | 1920x1080, landscape   |

Tips:
- Busy/high-contrast images will fight with the paper cards. Something
  textured and fairly dark/muted (an office desk, a real corkboard, a
  blurred open-plan office) reads best — the paper cards already carry the
  contrast.
- JPG or PNG both work. Keep files under ~1.5MB for fast loads.
